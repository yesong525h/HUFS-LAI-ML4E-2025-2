# 1. 해결하고자 하는 실제 문제

### 내가 겪고 있는 문제

수강하는 과목 중 하나인 **영어 디베이트 수업**을 준비할 때

* 주제 관련 자료(기사, 연구, 통계, 논문 등)를 읽는 데 시간이 오래 걸림
* 자료 속 주장이 찬성/반대 어느 쪽인지 빠르게 파악하기 번거러움
* 반박 포인트를 떠올리는 것도 시간이 많이 듦
* 상대 팀이 어떤 논리를 펼칠지 예측하기 어려움

이로 인해 준비 시간이 길어지고, 실제 디베이트에서 근거 부족 혹은 반박의 방향성을 생각하기 어려운 문제를 경험함.

### 왜 이 문제를 해결하고 싶은지

* 수업에서 더 설득력 있게 발언하고 싶음
* 매주 디베이트 준비 시간이 너무 길어 효율적으로 준비하고 싶음
* 자료를 요약·구조화하는 도구가 있으면 편리할 것이라 생각
* 상대가 할 주장을 미리 파악할 수 있으면 더 준비된 모습을 보여줄 수 있음.

### 실제로 사용할 것인지, 얼마나 사용할 예정인지

수업에서 매주 한두개 정도 주제의 토론 준비를 해야 하므로,
자연스레 5회 이상 사용할 에정.

# 2. 제안하는 솔루션

### 어떤 도구/서비스를 만들 것인지

입력된 자료(기사, PDF 텍스트, 논문 일부, 뉴스 등)를 분석하는 영어 디베이트 준비 보조 도구.

### 주요 기능

1.  **찬성/반대 stance 자동 분류**
   * 입력 텍스트의 문장들이 어느 쪽 입장인지 자동 파악

2. **문장 단위로 논지 분석 (주장, 근거 등으로 구분)**

3. **상대측 주장으로 활용될 수 있는 부분 추출(상대측 주장 예측)**
    
4. **그에 상응하는 반박 포인트 생성**

5. **요약 기능**
   * 긴 자료를 디베이트용 핵심 요약으로 압축

### 내가 어떻게 사용할 것인지

* 디베이트 수업 준비하는 날
* 자료(뉴스, 기사, 브리핑 문서)를 복사해서 도구에 입력
* 도구가 제시하는 내용을 바탕으로 디베이트 발언 스크립트를 작성
→ 실제 디베이트 스크립트 품질과 준비 속도가 향상될것으로 기대.
