# 프로젝트 제안서: 영어 강의 자료 자동 번역 및 요약 도구

## 해결하고자 하는 문제 
전공 강의 자료(PDF, PPT 등)가 대부분 영어로 구성되어 있어, 시험 직전이나 복습 시 모든 자료를 한국어로 수동 번역해야 하는 반복 작업을 매 학기 겪는다. 
방대한 양의 자료를 번역하는 데 소요되는 시간으로 인해 실제 학습 시간이 줄어들며, 중요한 내용에 집중하기 어렵다. 
또한, 번역 후에도 핵심 문장이나 개념을 찾기 위해 다시 전체 텍스트를 읽어야 하는 비효율성이 발생한다.

강의 자료를 빠르게 한국어로 전환하고, 가장 중요한 핵심 문장만 요약하여 시험 준비 시간을 단축하고 학습 성과를 높이고 싶다. 
이를 통해, 영어 자료에 대한 장벽 없이, 핵심 내용에 즉각 집중하여 학습 효율을 극대화하고자 한다.

## 실제 사용 시나리오 
매 학기 시험 기간, 특히 중간/기말고사 2주 전부터 최소 10일 이상은 필수적으로 사용하여 전공 자료를 정리한다. 
사용자는 강의 자료의 영어 텍스트를 복사하여 도구에 입력하고, 도구가 제공하는 한국어 번역본과 핵심 요약을 기반으로 효율적인 시험 대비를 진행한다.

## 제안하는 솔루션
사용자가 직접 복사한 영어 텍스트를 입력하면, 자동으로 한국어 번역 및 핵심 문장 추출 요약을 제공하는 도구이다.

## 주요 기능 
- 단순 텍스트 입력: 사용자가 PDF나 PPT에서 직접 복사한 영어 텍스트를 스크립트에 입력한다.

- API 기반 번역: 기존 번역 API (Google Translate API 또는 공개된 LLM API)를 활용하여 한국어 번역 결과를 빠르게 얻는다.

- ML 기반 핵심 문장 추출: 번역된 한국어 텍스트에서 비지도 학습 기법을 활용하여 가장 중요한 문장들을 자동으로 추출한다.

- CLI 결과 출력: 원문, 번역문, 추출된 요약 문장 목록을 깔끔하게 출력한다.

## ML 활용 계획
**Assignment 4:**
- 클린 텍스트 데이터 수집: 전공 강의 자료에서 영어 텍스트 샘플을 직접 복사하여 클린 텍스트 파일(예: .txt)로 수집한다.
- 텍스트 전처리: 번역 API를 사용하여 수집된 영어 텍스트를 한국어 텍스트로 변환한다.
  
**Assignment 5:**
- 핵심 문장 추출 모델 학습 
  - 목표: 한국어 번역 텍스트에서 가장 중요한 문장들을 비지도 학습 방식으로 자동으로 선별한다.
  - 평가: 수동 평가를 통해 추출된 요약 문장이 원문의 핵심 내용을 얼마나 잘 포함하는지 확인한다. 또한, 추출된 문장의 압축률을 측정한다.
  - 활용 방안: 구현된 스크립트를 최종 도구에 통합하여 핵심 요약 기능을 제공한다.

## 기대 효과 
- 매 학기 전공 학습 자료 분석에 이 도구를 적극적으로 활용하여 학업 성취도 향상에 직접적인 도움을 얻을 수 있다.
- 영어 강의 자료의 번역 및 핵심 요약을 자동화하여 시험 준비 및 복습 시간을 획기적으로 단축하고, 내용 이해에 집중할 수 있다.
