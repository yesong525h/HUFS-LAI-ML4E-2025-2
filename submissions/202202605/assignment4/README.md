# 1. 데이터 수집
전체 과정은 ChatGPT로 생성한 코드를 기반으로 수행함

- 수업 자료(중남미경제, 문예사조의이해)와 온라인 뉴스 기사 등 총 5개의 텍스트를 수집
- konlpy(Okt)를 사용하여 명사·동사·형용사만 추출
- 각 텍스트를 개별 CSV 파일로 저장 후 최종적으로 하나의 데이터셋으로 병합

## 1.1. 전처리
- 한글 및 공백을 제외한 문자 제거
(숫자, 영문, 특수기호, 불필요한 기호 정리)
- 중복 단어 제거 및 정렬
- 최종적으로 총 1,244개의 고유 단어 데이터셋 확보

# 2. 수작접 라벨링
- 즉시 의미를 이해하는 단어는 1, 어렵거나 이해가 불확실한 단어는 0으로 직접 라벨링
- 최종 라벨 분포:
	- 1 : 1,018개
	- 0 : 226개 
- 전체 데이터 중 어려운 단어 비율을 약 18.1%로 확인됨, 라벨 불균형이 존재함.

# 3. EDA

## 3.1. 단어 길이 비교
- 라벨별 평균 단어 길이:
	- 0 : 평균 2.38
	- 1 : 평균 2.24
난이도가 높은 단어가 상대적으로 더 긴 경향을 보였으나 차이가 크지 않아 단독 feature로는 한계가 있음

## 3.2. 주요 관찰 결과
- 2글자 단어 내 난이도 차이가 뚜렷하게 확인됨
- 단어 길이만으로 난이도 예측을 수행하기 어렵다는 근거로 작용함	

# 4. 분석 결과

난이도는 단순한 길이 정보만으로 판단하기 어렵기 때문에, 추가적인 **언어적 특징(feature)**을 고려할 필요가 있는 것으로 확인되었다.
특히 다음과 같은 세 가지 요소가 난이도 차이를 설명하는 데 유의미하게 작용할 가능성이 있다.
- 자모 기반 패턴
단어를 의미 단위가 아닌 자모 수준에서 분석하는 방식으로, 받침 유무나 특정 음절 구성 등의 구조적 특징을 반영할 수 있음.
-  명사 중심 난도 차이.
난이도는 명사에서 가장 크게 구분되는 경향이 나타남.
예: 공시, 합작
- 빈도 기반 특징
	- 외부 말뭉치 빈도: 한국어 전체 사용 빈도를 통해 단어의 익숙함을 추정할 수 있음.
	- 텍스트 내부 빈도: 동일 자료에서 반복적으로 등장할수록 익숙해질 가능성이 높으며,
한 번만 등장하는 단어는 희귀 어휘일 가능성이 큼.

- 다층적 특징 결합의 필요성
단일 기준으로는 설명력이 제한되므로, 여러 특징을 동시에 활용하는 접근이 효과적일 것으로 보임.

예) 명사 + 받침 포함 + 특정 접미사(-성·-화 등) + 저빈도
와 같은 조합은 높은 난이도를 보일 가능성이 있다.
