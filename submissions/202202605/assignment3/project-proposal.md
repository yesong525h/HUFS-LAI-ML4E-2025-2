# 각종 문서 난이도 분석 및 어려운 단어의 뜻풀이와 스페인어 번역을 제공하는 도구

## 1. 해결하고자 하는 문제

해외에서 태어나 자라면서 스페인어를 모국어처럼 사용해왔기 때문에, 한국어로 된 자료를 읽을 때 종종 어려움을 겪는다. 대학교 수업에서 제공되는 자료나 한국어 뉴스 기사처럼 전문적이고 난이도가 높은 문서를 접할 때, 이해가 끊기거나 내용을 빠르게 따라가지 못하는 경우가 잦다.

특히, 교육 과정을 대부분 스페인어로 진행해 왔기 때문에, 배운 개념조차 한국어로는 익숙하지 않는 경우가 발생한다. (스페인어로는 알고 있는 용어라도, 한국어 표현을 모르면 문맥 이해가 어려워진다). 모르는 단어가 나올 때마다 매번 사전을 찾아보는 과정은 글의 흐름을 끊어 효율적인 학습에 방해된다.

## 2. 제안하는 솔루션

한국어 텍스트를 입력받아 문서의 전체적인 난이도를 예측하고, 모를 가능성이 높은 단어를 자동 선별하여 각 단어에 대한 쉬운 한국어 풀이와 스페인어 번역을 제공한다. 

### 주요 기능:

- 텍스트 전체 난이도 분석
- 사용자에게 어려운 가능성이 높은 단어 목록 자동 추출
	- 단어별 쉬운 한국어 풀이 제공
	- 스페인어 번역 제공
