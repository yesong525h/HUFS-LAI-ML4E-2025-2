## 2. 해결하고자 하는 문제

학교 생활을 하다 보면 여러 일정 때문에 시험 대비가 늦어지는 경우가 많습니다.  
하지만 교수님들이 배포하신 PPT 및 PDF 강의 자료는 방대한 양의 텍스트로 이루어져 있어,  
단기간에 핵심 내용을 정리하기가 쉽지 않습니다.

특히 다음과 같은 상황에서 어려움이 발생합니다:

- 긴 문장 위주의 PPT (예: AI공학, 어문 계열)  
- 도표나 개념 중심의 PDF 강의 자료  
- 외국어(특히 영어)로 작성된 수업 자료  

> **핵심 문제:**  
> 시험 전 짧은 시간 안에 *핵심 내용 파악 + 번역 + 요약 정리*를 빠르게 수행할 방법이 부족하다.

따라서, 수업 자료(PPT·PDF)를 업로드하면 자동으로 **핵심 내용 요약 + 번역 + 간단한 설명**을 제공하는 도구가 필요합니다.  
이 도구는 시험 대비뿐 아니라 발표 준비, 리포트 작성, 외국어 학습에도 활용 가능합니다.

---

## 3. 제안하는 솔루션

### 3.1 서비스 개요
**BonCahier AI**는 학습자가 PPT나 PDF 강의 자료를 업로드하면,  
AI가 자동으로 **핵심 요약, 번역, 설명**을 생성하여 복습 효율을 높이는 서비스입니다.

### 3.2 주요 기능
1. **파일 업로드:** `PPT`, `PDF` 파일 업로드 지원  
2. **텍스트 추출:** `PyMuPDF` 및 `python-pptx`를 통한 페이지·슬라이드별 텍스트 분석  
3. **핵심 요약 및 번역:** 문단별 요약 + 외국어의 경우 한국어 번역본 제공  
4. **자동 설명:** 요약된 내용을 기반으로 핵심 개념 및 간단한 예시 생성  
5. **결과 출력:** Markdown / PDF / TXT 형태로 저장 가능  

### 3.3 사용 시나리오
1. 기말고사 2주 전, “`ml4e-lecture-week13.pdf`” 업로드  
2. 서비스가 자동으로 각 슬라이드의 핵심 문장을 요약하고 번역 제공  
3. AI가 개념 해설 추가 (“비지도 학습은 정답 데이터 없이 패턴을 학습하는 방식...”)  
4. 결과를 “시험 대비 핵심 요약 노트”로 다운로드하여 복습

## 4. ML 활용 계획

### 4.1 Assignment 4 – 데이터 수집 및 분석
- 공개 요약 데이터셋 수집:
  - daekeun-ml/naver-news-summarization-ko (한국어: 네이버 뉴스 본문 + 요약)
  - gopalkalpande/bbc-news-summary (영어: BBC 뉴스 기사 + 요약)
- 수집한 데이터의 기본 정보 확인(df.info, describe)
- 데이터 길이/분포/요약 관계 분석(EDA)
- EDA 결과를 기반으로 모델 학습을 위한 전처리 방향 수립

### 4.2 Assignment 5 – 모델 학습 및 평가
- **모델 후보 비교**

| 모델 | 장점 | 단점 | 활용 계획 |
|------|------|------|-----------|
| **KoBART** | 한국어 요약 특화 | 영어 데이터에 약함 | 한글 강의 요약용 |
| **T5-small** | 다국어 지원 | 계산량 많음 | 번역+요약 통합 실험 |
| **Pegasus** | 문장 수준 요약 우수 | 데이터 필요량 많음 | 영문 슬라이드 요약용 |

- **학습 목표:** 문단 수준 핵심 요약 및 개념 설명 생성  
- **평가 지표:** ROUGE 점수 + 주관적 요약 품질 평가  

### 4.3 Assignment 6 – 실제 사용 및 개선
- 본인 수업 자료(PPT·PDF) 3~5개 업로드 후 테스트  
- 요약 및 번역 품질 평가, UI 개선  
- 사용자 피드백(본인 기준)을 반영하여 출력 포맷 수정

## 5. 기술적 고려사항

- **텍스트 추출:** `PyMuPDF` (PDF), `python-pptx` (PPT)  
- **ML 모델 학습:** `transformers`, `datasets` (Hugging Face)  
- **웹 UI:** `Flask` 또는 `Streamlit`  
- **출력 저장:** Markdown → PDF 변환  
- **번역 API 후보:** `DeepL API`, `Google Translate API`, 또는 `Helsinki-NLP/opus-mt`

## 6. 기대 효과 및 활용 계획

- 시험 대비 핵심 요약 노트를 빠르게 생성 가능  
- 강의 자료 복습 및 외국어 이해 효율 증대  
- 학기 중 최소 5회 이상 사용 가능 (중간·기말 대비, 발표 준비 등)  
- 장기적으로는 “**다국어 강의 자동 요약·번역 서비스**”로 발전 가능
